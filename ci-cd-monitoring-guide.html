🔍 Hướng dẫn xem CI/CD chạy

📍 Trên GitHub Web Interface

  1. Vào repository GitHub của bạn
  2. Click tab "Actions"
  3. Bạn sẽ thấy danh sách workflows
  4. Click vào workflow đang chạy để xem chi tiết
  5. Click vào job để xem logs real-time

⚡ Các trạng thái workflow

  • 🟡 Queued: Đang chờ runner
  • 🔵 In progress: Đang chạy
  • ✅ Success: Hoàn thành thành công
  • ❌ Failed: Thất bại
  • ⚪ Cancelled: Bị hủy

🔧 Trigger CI/CD

CI/CD sẽ tự động chạy khi:

  • Push code lên branch main/master/develop
  • Tạo Pull Request
  • Trigger thủ công từ Actions tab

📱 Mobile/Notifications

  • Install GitHub Mobile app
  • Enable notifications cho repository
  • Nhận thông báo khi workflow complete

🎯 Xem kết quả

  • Test Results: Trong "Summary" của workflow
  • Artifacts: Download từ workflow run
  • GitHub Pages: Test reports dashboard
  • Logs: Chi tiết từng step
